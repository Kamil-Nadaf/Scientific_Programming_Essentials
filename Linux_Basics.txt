Going inside the directory
cd directory_name

Coming out of that directory 
cd..

Coming out of all sub-directories
cd 

Getting path of working directory
pwd

Copying a file to path 
cp file_name.extension path

Moving file to path
mv file_name.extension path

Renaming file
mv old_file_name new_file_name

Moving all files with same extension (.py) to different path
mv *.py path
